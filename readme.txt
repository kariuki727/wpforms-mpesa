== Description ==  
This plugin integrates M-Pesa payments with WPForms, allowing users to make payments via M-Pesa when submitting forms.  

== Features ==  
- Secure integration with M-Pesa Daraja API  
- Supports Lipa na M-Pesa STK Push  
- Automatic payment confirmation via callback  
- Displays payment status & transaction ID in WPForms entries  

== Installation ==  
1. Install and activate WPForms (if not already installed).  
2. Upload `wpforms-mpesa.zip` via Plugins → Add New → Upload Plugin.  
3. Go to **Settings → WPForms M-Pesa** and enter your M-Pesa API credentials.  
4. Add an M-Pesa payment field to your WPForms form.  
5. Start receiving payments via M-Pesa!  

== Frequently Asked Questions ==  

= Does this work with WPForms Lite? =  
Yes! WPForms M-Pesa works with both WPForms Lite and WPForms Pro.  

= Do I need a Safaricom Developer Account? =  
Yes. You need an M-Pesa Daraja API account to get the consumer key & secret.
